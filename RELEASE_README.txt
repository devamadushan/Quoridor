QUORIDOR GAME - Windows Release v1.0.2
==========================================

INSTALLATION:
1. Extract this ZIP file to any folder on your computer
2. The game is ready to run - no additional installation required!

HOW TO RUN:
- Double-click "Quoridor.exe" in the extracted folder
- Or run from command line: .\Quoridor.exe

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- No Java installation required (bundled with the game)

WHAT'S INCLUDED:
- Quoridor.exe - The game executable
- app/ - Game files and dependencies
- runtime/ - Bundled Java runtime environment

GAME FEATURES:
- Classic Quoridor strategy board game
- Intuitive graphical interface built with JavaFX
- Single-player and multiplayer modes
- Beautiful animations and modern UI

TROUBLESHOOTING:
- If the game doesn't start, try running as administrator
- Make sure Windows Defender/antivirus isn't blocking the executable
- Keep all files in the same folder - don't move Quoridor.exe separately
